¿Qué es GitHub?

GitHub es un servicio basado en la nube que aloja un sistema de control de versiones (VCS) llamado Git. Éste permite a los desarrolladores colaborar y realizar cambios en proyectos compartidos, a la vez que mantienen un seguimiento detallado de su progreso; Git es un proyecto de código abierto que se inició en 2005 y creció hasta convertirse en uno de los VCS más populares del mercado Git es un proyecto de código abierto que se inició en 2005 y creció hasta convertirse en uno de los VCS más populares del mercado

Tom Preston-Werner

Es un multimillonario estadounidense desarrollador de software y empresario. Es un colaborador activo dentro del comunidad de desarrollo de código abierto Es mejor conocido como el fundador y ex CEO de GitHub, a Git repositorio servicio de hospedaje basado en web, que cofundó en 2008 con Chris Wanstrath y PJ Hyett. Renunció a GitHub en 2014 cuando una investigación interna concluyó que él y su esposa acosaron a un empleado.

PJ Hyett

Es un desarrollador de software y emprendedor que cofundó GitHub en 2008, junto con Chris Wanstrath y Tom Preston-Werner, para simplificar el intercambio de código y facilitar la colaboración en la creación de software. En la actualidad, GitHub es el host de código más grande del mundo, con una comunidad de cinco millones de personas que crean software juntas.

Chris Wanstrath

Chris Wanstrath es cofundador de GitHub, el servicio de alojamiento y colaboración de software más grande del mundo. Antes de fundar GitHub, trabajó como ingeniero en CNET Networks en Gamespot y en el lanzamiento de Chow. Chris se mudó a San Francisco en 2005 después de abandonar la universidad.

Scott Chacon

Trabaja en Chatterbug (la mejor manera de aprender un idioma) y anteriormente ayudó a iniciar GitHub. Scott Chacon es un desarrollador de software, evangelista de Git, orador, escritor, viajero mundial, padre, esposo, salvador de gatos, signatario de bebés y domador de gorilas.
